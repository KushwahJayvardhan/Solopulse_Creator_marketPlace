Creator Marketplace

Creator Marketplace is a platform designed to connect content creators, marketers, brands, and companies. It enables users to build rich profiles, showcase portfolios, and discover collaboration opportunities aligned with their goals. The backend is built with Spring Boot and follows clean, modular architecture principles for scalability and maintainability.

Key Features

Multi-role user onboarding for creators, brands, marketers, and companies

Profile management with detailed attributes and content formats

Portfolio creation using JPA entity relationships

Discovery and matchmaking workflows for collaborations

Spring Boot backend with Spring Data JPA and Hibernate

PostgreSQL integration with clear schema structure

Extensible design for future modules including messaging, payments, and analytics
