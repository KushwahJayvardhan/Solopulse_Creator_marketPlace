Solopulse – Creator Marketplace Platform
Overview

Solopulse is a creator marketplace platform designed to connect content creators, brands, marketers, and companies for meaningful collaborations. The platform focuses on profile-based discovery, collaboration requests, messaging, and trust signals to enable efficient and transparent partnerships.

This project is currently focused on building a strong backend foundation with clean domain modeling and scalable architecture.

Core Objectives

Enable creators, brands, marketers, and companies to create and manage profiles

Facilitate collaboration and campaign-based engagements

Provide secure messaging between users

Establish reputation and trust mechanisms for better decision-making

User Roles

Creators: Showcase content, portfolio, pricing, and availability

Brands: Find creators for promotions and campaigns

Marketers: Manage campaigns and connect with creators and brands

Companies: Hire marketers or collaborate with creators at scale

Key Features

Role-based profile management

Creator portfolio and content format support

Collaboration and campaign requests

Direct messaging system

Project and booking status tracking

Extensible trust and reputation model

Tech Stack

Language: Java 17

Framework: Spring Boot

ORM: JPA / Hibernate

Database: PostgreSQL

Build Tool: Maven

Version Control: Git & GitHub
